# Odin Recipes 🍽️

A simple recipe website built as part of [The Odin Project](https://www.theodinproject.com/) curriculum.

## About

This project is a basic HTML recipe site with no CSS styling (yet). It focuses on practicing core HTML structure and elements including headings, links, images, lists, and proper HTML boilerplate.

## Project Structure

```
odin-recipes/
├── index.html
└── recipes/
    ├── artichokeSpinachLasagna.html
    ├── mushroomStirFry.html
    └── zucchiniRollatini.html
```

## Features

- Home page (`index.html`) with links to all recipes
- Individual recipe pages containing:
  - Recipe name as an `h1` heading
  - Image of the finished dish
  - Description section
  - Ingredients (unordered list)
  - Steps (ordered list)
  - Navigation link back to the home page

## Built With

- HTML5 boilerplate
- Semantic HTML elements

## What I Learned

- Setting up a proper HTML file structure
- Linking between multiple HTML pages using relative paths
- Using semantic elements: headings, paragraphs, images, and lists
- Organizing a multi-page project with a logical folder structure
