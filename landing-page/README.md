# Landing Page

A simple landing page built as part of The Odin Project Foundations curriculum.


## About

This project involved building a full landing page from a design mockup using only HTML and CSS. The focus was on applying Flexbox to create a structured, multi-section layout.

## Built With

- HTML5
- CSS3 (Flexbox)

## What I Learned

- Using Flexbox to arrange and align page sections
- Structuring an HTML page with semantic elements
- Translating a visual design into code
